在markdown的語法裏面
有兩種創造出標題的方法ww

`Setext`和`atx`
Setext
=========
就是在標題下面加上'=' 或 '-'

像這樣 
 (第一階標題)
```
我是標題owo
========== 
```
>*想要加多少'='應該都是可以的www*

和
 (第二階標題)
```
我是標題owo
----------
```

# atx
在行首插入`1~6`個`#`
像這樣
```
# 我是標題.w. 超大的那種
## uwu
### owo
#### >w<
##### QwQ
###### AwA
```
標題後面加上'#'也是可以正常使用的
像是
```
# owo #
## uww ##
### >w< ##############
```
*`#`想加多少個都沒問題*
